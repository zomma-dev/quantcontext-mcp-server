# QuantContext

Deterministic quant tools for AI trading agents. An MCP server for trade research and strategy backtesting.

QuantContext gives Claude, GPT, and any MCP-compatible AI agent real quantitative computation — stock screening, backtesting, and factor analysis. Every number is computed from historical market data, not generated by an LLM.

## Install (30 seconds)

```bash
pip install quantcontext-mcp
```

Then add to your AI agent:

**Claude Code:**
```bash
claude mcp add quantcontext -- quantcontext
```

**Claude Desktop** (`claude_desktop_config.json`):
```json
{
  "mcpServers": {
    "quantcontext": {
      "command": "quantcontext"
    }
  }
}
```

No API keys. No configuration. The first query works immediately.

## What It Does

Three tools that compose into a complete strategy research workflow:

| Tool | What It Does |
|------|-------------|
| `screen_stocks` | Filter S&P 500, Russell 2000, or Nasdaq 100 by fundamental, momentum, value, quality, technical, or multi-factor criteria. Returns ranked candidates with scores. |
| `backtest_strategy` | Test a screening strategy historically. Rebalance-loop engine with daily P&L tracking, position sizing (equal-weight or inverse-vol), stop-loss, and max-drawdown circuit breaker. Returns equity curve, trade log, and metrics (CAGR, Sharpe, max drawdown, Calmar, win rate). |
| `factor_analysis` | Decompose strategy returns into Fama-French factors (market, size, value, momentum) using OLS regression. Returns alpha with t-statistic, factor loadings, R-squared. |

Tools compose naturally: **screen** stocks, **backtest** the screen as a strategy, then run **factor analysis** to understand where the returns come from.

## Quick Example

**You:** "Screen S&P 500 for value stocks with PE under 15 and ROE above 12%"

**Agent calls** `screen_stocks`:
```json
{
  "universe": "sp500",
  "screen_type": "fundamental_screen",
  "config": {"pe_lt": 15, "roe_gt": 12}
}
```

**Result:** 23 stocks found. Top picks: VTRS (PE 7.2, ROE 18%), MOS (PE 8.1, ROE 22%), BG (PE 9.4, ROE 16%).

**You:** "Backtest those value stocks — monthly rebalance, equal weight, 2 years"

**Agent calls** `backtest_strategy` with the same screen criteria. Returns: +32.4% total return, 1.24 Sharpe, -14.8% max drawdown.

**You:** "Where is the alpha coming from?"

**Agent calls** `factor_analysis` on the equity curve. Returns: 4.7% annualized alpha (t=1.87), high value exposure (HML loading 0.49), moderate size tilt (SMB 0.22).

Three tools. One conversation. Every number computed from real market data.

## Screen Types

| Screen | Parameters | Description |
|--------|-----------|-------------|
| `fundamental_screen` | `pe_lt`, `roe_gt`, `debt_equity_lt`, `revenue_growth_gt` | Filter by PE, ROE, leverage, growth |
| `quality_screen` | `roe_gt`, `debt_equity_lt`, `profit_margin_gt` | High-quality companies |
| `momentum_screen` | `lookback_days`, `top_pct` | Rank by N-day price momentum |
| `value_screen` | `pe_lt`, `top_n` | Cheapest stocks by valuation |
| `factor_model` | `weights` (value/momentum/quality/volatility), `top_n` | Multi-factor composite ranking |
| `technical_signal` | `rsi_period`, `sma_short`, `sma_long` | RSI and SMA crossover signals |
| `mean_reversion` | `lookback_days`, `z_threshold` | Z-score below threshold |

## Data Sources

- **Prices:** yfinance (free, no API key required)
- **Fundamentals:** yfinance (PE, ROE, debt/equity, margins, revenue growth)
- **Factor data:** Kenneth French Data Library (Fama-French 3 factors + momentum)
- **Universes:** S&P 500, Russell 2000, Nasdaq 100 (constituent lists bundled)

All data is cached locally (~50MB for full coverage). First query downloads data (~10s), subsequent queries are instant.
