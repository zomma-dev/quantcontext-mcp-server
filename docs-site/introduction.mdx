QuantContext is an [MCP server](https://modelcontextprotocol.io/) that turns plain-English strategy descriptions into executable quant research: screen stocks by any criteria, backtest over historical data, and run factor analysis to see where the returns come from. Every number is **computed from real market data**, not generated by an LLM. Results are fully reproducible.

Works with Claude, Codex, OpenCode, or any other MCP-compatible coding agent.

## The three tools

<CardGroup cols={3}>
  <Card title="screen_stocks" icon="filter" href="/tools/screen-stocks">
    Filter a stock universe by fundamentals, momentum, quality, technical signals, or a multi-factor blend. Returns ranked candidates.
  </Card>
  <Card title="backtest_strategy" icon="chart-line" href="/tools/backtest-strategy">
    Test a strategy over history with a rebalance-loop engine. Returns CAGR, Sharpe, max drawdown, equity curve, and trade log.
  </Card>
  <Card title="factor_analysis" icon="magnifying-glass-chart" href="/tools/factor-analysis">
    Decompose strategy returns into Fama-French factors (market, size, value, momentum). Returns alpha with t-statistic.
  </Card>
</CardGroup>

## How it works

Describe a strategy. The agent confirms the exact parameters it will use, then executes:

```
screen_stocks → backtest_strategy → factor_analysis
```

Screen candidates, backtest over history, decompose returns to see where the alpha comes from.

## Data sources

QuantContext uses free, public data only. No API keys required.

| Data | Source | Details |
|------|--------|---------|
| **Price data** | Yahoo Finance (`yfinance`) | Daily OHLCV, 3+ years of history |
| **Fundamentals** | Yahoo Finance | PE ratio, ROE, debt/equity, revenue growth, margins |
| **Universe lists** | Wikipedia | S&P 500 and Nasdaq 100 scraped from constituent tables |
| **Factor data** | [Kenneth French Data Library](https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/data_library.html) | Daily Fama-French factors (Mkt-RF, SMB, HML, Mom) back to 1926 |

All data is cached locally at `~/.cache/quantcontext/` after first download. To seed the cache instantly instead of building it from scratch, run `quantcontext-warmup --url https://quantcontext.ai/api/data` once after install.

## Why determinism matters

LLMs hallucinate numbers. QuantContext enforces a strict boundary:

- **LLM layer**: decides *what* to compute, selects parameters, explains results in natural language
- **QuantContext layer**: executes deterministic Python computation, returns structured JSON
- **Data layer**: yfinance (prices + fundamentals), Kenneth French (Fama-French factors)

The LLM never touches a number. Python computes everything.
